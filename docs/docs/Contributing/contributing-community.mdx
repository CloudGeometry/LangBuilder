There are several ways you can interact with the Langbuilder community and learn more about the Langbuilder codebase.

## Join the Langbuilder Discord server

Join the [Langbuilder Discord Server](https://discord.gg/EqksyE2EX9) to ask questions and showcase your projects.

## Follow Langbuilder on X

Follow [@langbuilder_ai](https://twitter.com/langbuilder_ai) on X to get the latest news about Langbuilder.

## Star Langbuilder on GitHub

If you like Langbuilder, you can star the [Langbuilder GitHub repository](https://github.com/cloudgeometry/langbuilder).
Stars help other users find Langbuilder more easily, and quickly understand that other users have found it useful.

Because Langbuilder is open-source, the more visible the repository is, the more likely the codebase is to attract [contributors](/contributing-how-to-contribute).

## Watch the GitHub repository

You can watch the [Langbuilder GitHub repository](https://github.com/cloudgeometry/langbuilder) to get notified about new releases and other repository activity.

To get release notifications only, select **Releases only**.

If you select **Watching**, you will receive notifications about new releases as well as issues, discussions, and pull requests, if you are interested in that activity.
For information about customizing repository notifications, see the [GitHub documentation on repository subscriptions](https://docs.github.com/en/account-and-profile/managing-subscriptions-and-notifications-on-github/managing-subscriptions-for-activity-on-github/viewing-your-subscriptions).

## Request enhancements and get help through GitHub

You can also submit feature requests and get help with Langbuilder through the GitHub repository.
For more information, see [Get help and request enhancements](/contributing-github-issues).
